# Agent Plugins for Clay

Build with Clay in your AI coding agent — skills, MCP tools, and the Clay CLI.

## Installation

### Claude Code

```
/plugin marketplace add clay-run/agent-plugins
/plugin install clay
```

### Codex

Run `/plugins`, add `clay-run/agent-plugins`, then install **clay**.

### Cursor

Search **clay** in the [Cursor marketplace](https://cursor.com/marketplace) and click Install.
